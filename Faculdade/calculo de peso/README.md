- Sobre o projeto

Este projeto foi desenvolvido em Python com o objetivo de realizar cálculos relacionados ao peso de forma simples e eficiente. Ele permite ao usuário obter informações úteis como cálculo de peso ideal, índice de massa corporal (IMC) ou outras medidas relacionadas.

O foco do projeto é ser fácil de usar e servir como prática para quem está iniciando na programação.

- Funcionalidades

Cálculo de peso ideal
Cálculo de IMC
Entrada de dados pelo usuário
Exibição de resultados no terminal

- Tecnologias utilizadas

Python 3
