# Cálculo de Gasto de Combustível em C

Este projeto foi desenvolvido como exercício de programação utilizando a linguagem **C**.  
O objetivo do programa é calcular o gasto estimado de combustível de uma viagem.

preview:

</p>

Objetivo

Criar um programa que ajude um motorista a saber quanto irá gastar com combustível em uma viagem, a partir da distância e do preço do litro da gasolina.

Como funciona

O programa solicita ao usuário duas informações:

1. A distância da viagem em quilômetros (km)
2. O preço do litro do combustível

Após receber os dados, o programa realiza o cálculo e exibe o gasto estimado da viagem.

Tecnologias utilizadas

- Linguagem C
- Compilador GCC
- Visual Studio Code

Exemplo de uso

Entrada:

Distância da viagem: 100 km  
Preço do combustível: 5.50

Saída:

Gasto estimado da viagem: R$ 55.00
