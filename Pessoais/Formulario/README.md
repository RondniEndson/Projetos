- Formulário Web Interativo

Este projeto consiste em um formulário web desenvolvido utilizando **HTML**, **CSS** e **JavaScript**, com foco em uma interface simples, responsiva e validação de dados no lado do cliente.

-Funcionalidades

* Estruturação semântica com HTML5
* Estilização moderna e responsiva com CSS3
* Validação de campos utilizando JavaScript
* Feedback visual para o usuário (erros e sucesso)
* Layout adaptável para diferentes dispositivos

- Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript 

- Estrutura do Projeto

formulario-web
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┗ 📄 script.js

Como Usar

 Abra o arquivo `index.html` no seu navegador.



Validações Implementadas

* Campos obrigatórios
* Verificação de e-mail válido
* Limite de caracteres
* Mensagens de erro personalizadas
